# نظام Waheed - Restaurant OS

## ما هو هذا المشروع؟
نظام إدارة مطاعم ذكي يعمل بالذكاء الاصطناعي.

## هيكل المشروع
- backend/api        → FastAPI (Python) - السيرفر
- backend/agents     → AI Agents - وكلاء الذكاء الاصطناعي  
- backend/database   → قاعدة البيانات
- frontend/          → Next.js - واجهة الكاشير

## التقنيات المستخدمة
- Python + FastAPI → Backend
- Next.js + React → Frontend
- PostgreSQL → قاعدة البيانات
- Claude API → الذكاء الاصطناعي

## قواعد مهمة
1. كل كود يجب أن يكون بالإنجليزية
2. التعليقات بالعربية مسموحة
3. كل function لها مسؤولية واحدة فقط
4. لا تكسر ما يعمل
